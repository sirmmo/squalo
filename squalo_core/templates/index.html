Squalo [/'skwalo/] is the italian word for Shark. The pun is obvious: Shark eats stuff, Shark eats SQL file, SQL similar to Squalo. Squalo eats SQLite files!

Squalo helps you easily create an API around a SQLite database, wherever it comes from.
